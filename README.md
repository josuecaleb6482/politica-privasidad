# Política de Privacidad

Sitio web con las políticas de privacidad de las aplicaciones desarrolladas por **josuecaleb6482**.

## Acceso

La política de privacidad está disponible en:

**[https://josuecaleb6482.github.io/politica-privasidad/](https://josuecaleb6482.github.io/politica-privasidad/)**

## Contenido

El archivo `index.html` contiene la política de privacidad completa en español, que cubre:

- Información recopilada por las apps
- Cómo se usa esa información
- Compartición con terceros
- Servicios de terceros utilizados (Google Play Services, Firebase, etc.)
- Seguridad de la información
- Privacidad de menores
- Derechos del usuario
- Contacto

## Activar GitHub Pages

Para publicar este sitio activa **GitHub Pages** en la configuración del repositorio seleccionando la rama `main` como fuente.
