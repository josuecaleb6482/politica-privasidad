# Política de Privacidad Adicional — EasyBilling

Esta página describe las políticas adicionales de privacidad aplicables a EasyBilling. Aquí se detallan las prácticas relacionadas con el manejo de datos y permisos específicos que no se encuentran en la política principal.

## Permisos Adicionales

- **ACCESS_FINE_LOCATION**: Este permiso se utiliza para proporcionar servicios basados en la ubicación, como la generación de facturas con datos geográficos. La ubicación no se comparte con terceros y se utiliza únicamente dentro de la aplicación.
- **BLUETOOTH**: Utilizado para conectar dispositivos periféricos como impresoras portátiles. No se recopila información adicional a través de esta conexión.

## Uso de Datos

- Los datos recopilados a través de estos permisos se procesan exclusivamente en el dispositivo del usuario.
- EasyBilling no almacena ni transmite datos personales fuera del dispositivo.
